Mixed Alkene–Alkyne Compounds

Some organic molecules contain both a carbon–carbon double bond (alkene) and a carbon–carbon triple bond (alkyne). This lesson explains how to identify, number, and name these compounds correctly.

1. What Are Mixed Alkene–Alkyne Compounds?

A mixed alkene–alkyne compound is a hydrocarbon that contains:

  • At least one C=C double bond
  • At least one C≡C triple bond

Example: One double bond and one triple bond in the same carbon chain.

2. Naming Priority: Alkene vs Alkyne

When both a double bond and a triple bond are present:

  • The compound uses the combined suffix -en-yne
  • The parent chain must include BOTH multiple bonds
  • The chain is numbered to give the lowest set of locants

Activity 1: Concept Check

True or False: The triple bond always gets priority over the double bond when numbering.

False.
Numbering follows the lowest set of locants rule. If there is a tie, the double bond (ene) gets the lower number.

3. Step-by-Step Naming Procedure

  1. Identify the longest chain containing both the C=C and C≡C bonds.
  2. Number the chain to give the lowest possible locants to the multiple bonds.
  3. If there is a tie, assign the lower number to the double bond.
  4. Name the parent chain using -en-yne.
  5. Include substituents if present.

Numbering gives priority to the lowest multiple-bond positions.

5. Practice Activities

Practice 1

Which suffix is used for a compound containing both a C=C and a C≡C bond?

-en-yne

Practice 2

If a double bond is at carbon 2 and a triple bond is at carbon 5, which bond is named first?

The double bond is named first: -2-ene-5-yne.

Practice 3

Why must the parent chain include both multiple bonds?

Because the parent chain defines the compound’s identity and must show all principal functional features.

Key Terms

  • Alkene: Hydrocarbon with a C=C double bond.
  • Alkyne: Hydrocarbon with a C≡C triple bond.
  • Mixed unsaturated compound: Contains both C=C and C≡C bonds.
  • -en-yne: IUPAC suffix for compounds with both alkene and alkyne groups.
  • Lowest locant rule: Numbering rule that minimizes bond positions.
