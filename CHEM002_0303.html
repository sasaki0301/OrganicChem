Resonance Structures

Resonance describes molecules where a single Lewis structure is not enough to show how electrons are truly arranged. Instead of one fixed picture, the molecule is best represented as a “blend” of multiple valid structures.

1. What is Resonance?

Some molecules cannot be accurately represented by only one Lewis structure. When electrons can be placed in more than one valid arrangement, each arrangement is called a resonance structure.

The true structure is a resonance hybrid — a combination (or average) of all valid resonance structures.

Example: Nitrite ion (NO₂⁻) has two resonance structures with the double bond on either oxygen.

Activity 1: Concept Check

Does resonance mean a molecule is “flipping back and forth” between structures?

No. Resonance structures are not changing forms. The molecule is best described as a mixture (hybrid) where electrons are delocalized, not switching.

2. When Do We Use Resonance Structures?

Resonance is applied when:

  • There are multiple possible placements of double bonds.
  • Lone pairs can form a double bond (as long as octet is not violated).
  • A charge can be placed on more than one atom.
  • A π bond (double or triple bond) can shift position.

Resonance is never used to move atoms — only electrons.

Activity 2: Valid or Invalid?

In a resonance structure, which of the following is allowed?

  • A) Move a π bond to a different location
  • B) Move a lone pair to form a π bond
  • C) Move atoms to a new location
A and B are allowed (moving electrons). C is not allowed — atoms never move.

Only B is correct. σ bonds and atom positions must stay fixed.

4. Major vs Minor Resonance Contributors

Some resonance structures contribute more than others to the hybrid. A major contributor is more stable and therefore weighted more heavily.

Major structure characteristics:

  • Full octets on all atoms
  • Negative charge on the more electronegative atom
  • Fewer formal charges
  • Charges separated as little as possible

Activity 4: Select the Major Contributor

Which structure is more stable?

  1. Structure A: Oxygen has the negative charge
  2. Structure B: Carbon has the negative charge
Structure A is more stable because oxygen is more electronegative and stabilizes the negative charge.

5. Example: Resonance in Ozone (O₃)

Ozone contains one double bond and one single bond, but these bonds shift positions.

The resonance hybrid has equal O–O bond lengths.

Activity 5: Explain the Hybrid

Why are both O–O bonds the same length in ozone?

Because the π electrons are delocalized over both O–O bonds. The true structure is a hybrid, so both bonds have identical partial double-bond character.

Key Terms

  • Resonance: Multiple valid Lewis structures representing delocalized electrons.
  • Resonance structure: One possible arrangement of electrons.
  • Resonance hybrid: The true structure — a blend of contributors.
  • Delocalization: Electrons spread across multiple atoms.
  • Major contributor: Most stable resonance structure.
  • Minor contributor: Less stable structure; contributes less.
  • Curved arrows: Show movement of electron pairs, not atoms.
