Naming Cycloalkanes – Advanced Practice

Now that you know the naming steps, this lesson focuses on challenging real-world examples. No step-by-step breakdowns - just naming, logic, and careful rule application.

Key Reminders

  • Number to give the lowest set of locants.
  • Use alphabetical order (ignore di-, tri-, tert-, sec-).
  • If the substituent chain is longer than the ring, the ring becomes a substituent.
  • 1-substituent cycloalkanes DO NOT require the number 1.

Advanced Practice (10 Problems)

Problem 1

1-ethyl-3-methylcyclohexane
Ethyl alphabetically outranks methyl, so it gets carbon 1.

Problem 2

1,3-dimethylcyclobutane
Lowest set of locants: 1 and 3.

Problem 3

1-bromo-2-chlorocyclopentane
Alphabetical order: bromo (B) before chloro (C).

Problem 4

1-isopropyl-3-methylcyclohexane
“Isopropyl” counts as I for alphabetical ordering.

Problem 5

1-tert-butyl-2-methylcyclopentane
Alphabetization uses **B** for butyl (ignore tert-).

Problem 6

1-ethyl-2,4-dimethylcyclohexane
Lowest set: 1,2,4 (not 1,3,5).

Problem 7

propylcyclopentane
The ring (cyclopentane) has greater number of carbon than the chain (5C ring vs 3C chain?). The ring is the parent.

Problem 8

1,2,4-trimethylcyclohexane
Lowest set rule: 1,2,4 beats 1,3,5.

Problem 9

ethylcyclohexane
A single substituent doesn’t require numbering.

Problem 10

1-bromo-3-chloro-2-methylcyclohexane
Alphabetical: bromo (B), chloro (C), methyl (M).
