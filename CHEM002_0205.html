Organic Compound Nomenclature

In this lesson, you’ll learn how to name organic compounds systematically using IUPAC rules — from simple hydrocarbons to compounds with substituents and functional groups.

1. Introduction to IUPAC Nomenclature

The International Union of Pure and Applied Chemistry (IUPAC) developed a standardized way to name organic compounds so that each name represents exactly one structure.

The IUPAC name consists of three main parts:

  • Prefix: shows substituents or branches.
  • Parent name: indicates the longest carbon chain.
  • Suffix: denotes the primary functional group.

2. Naming Alkanes (Saturated Hydrocarbons)

Alkanes contain only single (σ) bonds between carbon atoms. The root name is based on the number of carbons in the longest chain.

  • 1 – Meth-
  • 2 – Eth-
  • 3 – Prop-
  • 4 – But-
  • 5 – Pent-
  • 6 – Hex-, etc.

The suffix –ane indicates all single bonds.

Activity 1: Identify the Compound

What is the name of a 5-carbon chain alkane with no branches?

Answer: Pentane (C₅H₁₂)

3. Naming Branched Hydrocarbons

For branched alkanes, identify and number the longest carbon chain, then name the attached groups (alkyl substituents) as prefixes.

Use the lowest possible numbers for substituent positions.

Example: 2-methylpropane (isobutane)

Activity 2: Apply Numbering Rules

What is the correct IUPAC name for this structure?

Answer: 2-Methylbutane. The longest chain has 4 carbons (butane). A methyl group is attached to the second carbon.

4. Functional Group Suffixes

Functional groups determine the suffix used in the compound name:

  • –OH (alcohol): –ol
  • –CHO (aldehyde): –al
  • –COOH (carboxylic acid): –oic acid
  • –NH₂ (amine): –amine

Answer: Ethanol. The suffix “–ol” indicates an alcohol; two-carbon chain gives “eth-”.

5. Multiple Substituents and Complex Names

When there are multiple identical substituents, use prefixes like di-, tri-, or tetra-. List different substituents alphabetically, ignoring prefixes.
