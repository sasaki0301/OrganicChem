Organic Nomenclature — Step-by-Step Guide

This lesson teaches a reliable naming workflow for organic molecules. We’ll focus on IUPAC’s systematic steps — how to find the parent chain, identify substituents, and select the correct suffix. Detailed alkane-specific naming will appear in the next chapter.

1. Structure of an IUPAC Name

The IUPAC name of an organic compound has three essential parts:

  • Prefix – identifies substituents and their positions.
  • Parent name – indicates the longest carbon chain (root word).
  • Suffix – identifies the main functional group.

Common Parent Chain Prefixes

CarbonsPrefixExample
1meth-CH₄ → methane
2eth-C₂H₆ → ethane
3prop-C₃H₈ → propane
4but-C₄H₁₀ → butane
5pent-C₅H₁₂ → pentane
6–10hex-, hept-, oct-, non-, dec-6–10 carbon chains

2. Step-by-Step Procedure

  1. Find the longest continuous carbon chain (parent).
  2. Number the chain to give substituents the lowest possible locants.
  3. Identify substituents (alkyl, halo, nitro, etc.).
  4. Assign locants and combine using hyphens and commas.
  5. Add multiplicative prefixes (di-, tri-, tetra-).
  6. Select the suffix for the principal functional group.
  7. Combine everything: Prefix + Parent + Suffix.

3. Selecting the Correct Suffix

The suffix reflects the main functional group and determines how you number the chain. If multiple groups are present, the one with the highest priority defines the suffix.

  • –COOH → -oic acid
  • –CHO → -al
  • –OH → -ol
  • –NH₂ → -amine

Activity 2: Suffix Selection

Which suffix takes priority if both –OH and –COOH are in the molecule?

Answer: Use -oic acid (carboxylic acid) since it outranks –OH.

  2. Double bond starts at carbon 2 → "-2-ene"
  3. Final name: but-2-ene.

If both ends could be numbered, choose the direction that gives the double bond the lower locant.

Example 3: Multiple Substituents

Name this compound containing both fluoro and methyl substituents.

  1. Parent chain: 5 carbons → pent-
  2. Substituents: fluoro at C-2, methyl at C-3.
  3. Alphabetical order: fluoro before methyl.
  4. Final name: 2-fluoro-3-methylpentane.

5. Common Pitfalls and Tips

  • Always pick the longest chain that includes the main functional group.
  • Lowest locant rule: Number to give the smallest numbers to substituents or bonds.
  • Alphabetize substituents (ignore prefixes like di-, tri-, sec-, tert-).
  • Remember: double/triple bonds take precedence in numbering over simple substituents.
