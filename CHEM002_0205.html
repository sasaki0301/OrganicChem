Organic Nomenclature — Step-by-Step Guide

This lesson teaches a reliable naming workflow for organic molecules. We’ll focus on IUPAC’s (International Union of Pure and Applied Chemistry) systematic steps — how to find the parent chain, identify substituents, and select the correct suffix. Detailed alkane-specific naming will appear in the next chapter.

1. Structure of an IUPAC Name

The IUPAC name of an organic compound has three essential parts:

  • Prefix – identifies substituents and their positions.
  • Parent name – indicates the longest carbon chain (root word).
  • Suffix – identifies the main functional group.

2. Step-by-Step Procedure

Step Visual Guide / Example
1. Find the longest continuous carbon chain — this becomes the parent chain (root name).
Example: Longest chain = 5 carbons → “pent[a]-”
2. Number the chain from the end nearest to a substituent or the principal functional group to give the lowest possible locants.
Number so substituent positions are minimized.
3. Identify substituents (alkyl, halo, nitro, etc.) and note their positions on the chain.
Example: methyl groups on carbon 2 and 3
4. Assign locants and combine numbers with substituent names using hyphens and commas.
Result: “2-methyl and 3-methyl”
5. Add multiplicative prefixes (di-, tri-, tetra-) if identical groups repeat separating the locants with a comma.
Example: “2,3-dimethyl”
6. Select the suffix based on the principal functional group (–ane, –ol, –al, –oic acid, etc.).
Example: –OH → “–ol” → ethanol
For this one it's an alkane → “-ane”
7. Combine all parts — Prefix + Parent + Suffix → the final IUPAC name.
Example: 2,3-dimethylpentane

3. Selecting the Correct Suffix

The suffix reflects the main functional group and determines how you number the chain. If multiple groups are present, the one with the highest priority defines the suffix.

  • –COOH → -oic acid
  • –CHO → -al
  • –OH → -ol
  • –NH₂ → -amine

5. Common Pitfalls and Tips

  • Always pick the longest chain that includes the main functional group.
  • Lowest locant rule: Number to give the smallest numbers to substituents or bonds.
  • Alphabetize substituents (ignore prefixes like di-, tri-, sec-, tert-).
  • Remember: double/triple bonds take precedence in numbering over simple substituents.
