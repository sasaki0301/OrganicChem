Organic Nomenclature — IUPAC Workflow

This lesson gives a clear, repeatable IUPAC naming workflow (prefix + parent + suffix). We’ll cover the general steps and practice with multiple examples. Detailed alkane naming rules are covered in the next chapter.

1. Anatomy of an IUPAC Name

An IUPAC name typically has three parts:

  • Prefix — substituents (with locants and multiplicative prefixes)
  • Parent — longest continuous carbon chain (root)
  • Suffix — principal functional group (–ane, –ol, –al, –oic acid, –amine, etc.)

2. Step-by-step naming procedure

StepVisual guide / example
1. Find the longest continuous carbon chain
This chain is the parent (root) that determines the base name.
Example: longest = 5 C → pent-
2. Number the chain
Number from the end nearest the principal functional group or substituent to give lowest locants.
Number to minimize substituent locants
3. Identify substituents
Name alkyl, halo, nitro, etc., and record their locants.
Example: methyl groups at C-2 and C-3
4. Assign locants and combine
Use commas between numbers and hyphens between numbers and words (e.g., 2,3-dimethyl).
Example: 2,3-dimethyl
5. Use multiplicative prefixes
di-, tri-, tetra- when identical substituents repeat (do not alphabetize by these prefixes).
Example: 2,3-dimethyl
6. Choose the suffix
The suffix corresponds to the highest-priority functional group (–oic acid > –al > –ol > –amine > –ane).
Example: –OH ⇒ –ol
7. Assemble the name
Prefix (substituents with locants) + parent + suffix → finalize the IUPAC name.
Example: 2,3-dimethylpentane

3. Suffix priority rules (short list)

When multiple functional groups exist, the highest-priority functional group determines the suffix and numbering.

  • Carboxylic acids (–COOH) → suffix -oic acid
  • Aldehydes (–CHO) → suffix -al
  • Alcohols (–OH) → suffix -ol
  • Amines (–NH₂) → suffix -amine
  • If no functional group outranks alkanes, use -ane

5. Common pitfalls & quick tips

  • Always select the longest chain that contains the principal functional group.
  • Number the chain to give lowest possible locants to substituents and functional groups.
  • Use multiplicative prefixes (di-, tri-) for identical substituents but ignore them when alphabetizing.
  • Double/triple bonds affect numbering — give the unsaturation the lowest possible number.
  • When in doubt, draw alternative parent chains and compare locants.

Key Terms

  • Parent chain — the longest continuous carbon chain forming the root name.
  • Locant — the number indicating the position of a substituent or functional group.
  • Prefix — substituent identifier (with locant).
  • Suffix — indicates the principal functional group (determines numbering).
  • Multiplicative prefix — di-, tri-, tetra- for repeated substituents.
