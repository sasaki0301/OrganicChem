Alkyl Groups

Alkyl groups are the “side chains” or “branches” that attach to the main carbon chain in an organic molecule. Understanding them is essential before naming alkanes and drawing more complex structures.

1. What is an Alkyl Group?

An alkyl group is formed when one hydrogen atom is removed from an alkane. This leaves the carbon chain with an available bonding site (shown as R– or –R).

Examples:

  • Methane (CH₄) → Remove one H → methyl (CH₃–)
  • Ethane (CH₃–CH₃) → Remove one H → ethyl (CH₃–CH₂–)

Alkyl groups come from alkanes by removing one hydrogen.

Activity 1: Identify the Alkyl Group

If propane (C₃H₈) loses one hydrogen, what alkyl group is formed?

Propyl group: CH₃–CH₂–CH₂–

2. Naming Simple Alkyl Groups

The names of alkyl groups come from the parent alkane by replacing the ending –ane with –yl.

Common alkyl groups include:

  • Methyl → CH₃–
  • Ethyl → CH₃CH₂–
  • Propyl → CH₃CH₂CH₂–
  • Butyl → CH₃CH₂CH₂CH₂–

Activity 2: Convert to Alkyl Names

Give the alkyl names for these chains:

  • A) CH₃–
  • B) CH₃CH₂–
  • C) CH₃CH₂CH₂–
A) Methyl
B) Ethyl
C) Propyl

3. Branched Alkyl Groups

When the removed hydrogen is not from an end carbon, we get a branched alkyl group. These include isopropyl, isobutyl, sec-butyl, and tert-butyl.

Common branched alkyl groups:

  • Isopropyl → (CH₃)₂CH–
  • Isobutyl → (CH₃)₂CH–CH₂–
  • sec-Butyl → CH₃–CH₂–CH(CH₃)–
  • tert-Butyl → (CH₃)₃C–
