Alkenes: Cis–Trans and E–Z Designation

This lesson focuses on identifying and naming geometric isomers of alkenes using both cis–trans and the more general E–Z system. You will practice deciding which system to use and how to apply each correctly.

1. Why Alkenes Show Geometric Isomerism

Alkenes contain a C=C double bond. Unlike single bonds, double bonds cannot rotate freely. This locks substituents into fixed positions in space.

As a result, two different spatial arrangements may exist — these are called geometric (stereoisomers).

  • Same molecular formula
  • Same atom connectivity
  • Different spatial arrangement

Activity 1: Concept Check

Why don’t alkanes show cis–trans isomerism?

Alkanes contain only single bonds, which rotate freely. Without restricted rotation, fixed spatial arrangements cannot exist.

2. Cis–Trans Isomerism (Simple Cases)

The cis–trans system is used only when:

  • Each double-bonded carbon has one identical group
  • Only two different groups are being compared

cis = identical groups on the same side
trans = identical groups on opposite sides

Activity 2: Cis or Trans?

An alkene has a CH₃ group on each carbon of the double bond. If the CH₃ groups are on opposite sides, what is the designation?

Trans — identical CH₃ groups are on opposite sides.

3. Why Cis–Trans Is Not Always Enough

Cis–trans fails when all four substituents are different. In such cases, there is no clear “identical group” to compare.

To solve this, chemists use the E–Z system, which works for all alkenes.

Activity 3: Decide the System

An alkene has H, Cl, Br, and CH₃ attached to the double bond. Which naming system must be used?

The E–Z system must be used because all four substituents are different.

4. The E–Z System (Step-by-Step)

Step 1: Assign Priorities (CIP Rules)

Priority is based on atomic number. The atom with the higher atomic number gets higher priority.

  • Br > Cl > S > O > N > C > H

Step 2: Compare High-Priority Groups

  • Z (zusammen): high-priority groups on the same side
  • E (entgegen): high-priority groups on opposite sides

Activity 4: Priority Check

Which group has higher priority: –CH₂CH₃ or –CH₃?

–CH₂CH₃ has higher priority because the first point of difference involves a carbon bonded to another carbon instead of hydrogens.

5. Mixed Practice (Cis–Trans vs E–Z)

Practice 1

An alkene has H and CH₃ on each double-bonded carbon. Which naming system is appropriate?

Cis–trans — each carbon has one identical group.

Practice 2

An alkene has substituents H, F, Cl, and CH₃. The higher-priority groups are on the same side. What is the designation?

Z — highest-priority groups are on the same side.

Practice 3

True or False: An alkene can be both cis and Z at the same time.

True. When conditions allow cis–trans naming, the cis isomer often corresponds to Z.

Practice 4

Why is E–Z preferred in modern IUPAC naming?

Because it works for all alkenes and avoids ambiguity when four different groups are present.

Practice 5

Which designation corresponds to lower steric repulsion in most cases?

E (or trans) usually has lower steric repulsion because bulky groups are farther apart.

6. Extended Practice: Naming Alkenes with Cis–Trans and E–Z

For each problem below, determine: (1) the correct alkene name and (2) whether cis–trans or E–Z notation is required.

Practice 6.1

An alkene has the structure CH₃–CH=CH–CH₃ with both CH₃ groups on the same side.


Parent chain: 4 carbons → but-2-ene
Identical groups present → cis–trans allowed
Same side → cis-but-2-ene

Practice 6.2

CH₃–CH=CH–CH₃ with CH₃ groups on opposite sides.


Parent: but-2-ene
Opposite sides → trans-but-2-ene

Practice 6.3

An alkene has substituents H and Br on one carbon, and H and CH₃ on the other. Br and CH₃ are on the same side.


All four groups are different → use E–Z
Priorities: Br > H and CH₃ > H
High-priority groups on same side → Z
Parent chain: prop-1-ene
Name: (Z)-1-bromoprop-1-ene

Practice 6.4

CH₂=CH–CH₃


One carbon has two identical hydrogens → no geometric isomerism
Name: prop-1-ene

Practice 6.5

An alkene with substituents Cl and CH₃ on one carbon, and H and C₂H₅ on the other. Cl and C₂H₅ are opposite.


Use E–Z system
Priorities: Cl > CH₃ and C₂H₅ > H
Opposite sides → E
Parent: pent-2-ene
Name: (E)-2-chloropent-2-ene

Practice 6.6

An alkene named (Z)-hex-3-ene. What does the Z tell you about the substituents?

The highest-priority substituents on each double-bonded carbon are on the same side of the C=C bond.

Practice 6.7

CH₃–CH=C(CH₃)–CH₂–CH₃


Parent: pent-2-ene
All four substituents different → E–Z required
Highest priority groups opposite → E
Name: (E)-3-methylpent-2-ene

Practice 6.8

True or False: cis-but-2-ene and (Z)-but-2-ene describe the same molecule.

True.
In simple cases where cis–trans applies, cis usually corresponds to Z.

Practice 6.9

An alkene has four different substituents, but the name given uses cis–trans. Is this correct?

No.
Cis–trans cannot be used when all four substituents are different. E–Z must be used instead.

Practice 6.10

Which is generally more stable: (E)-hex-3-ene or (Z)-hex-3-ene? Explain why.

(E)-hex-3-ene is more stable because bulky substituents are farther apart, reducing steric repulsion.

Key Terms

  • Geometric isomers: Isomers differing in spatial arrangement
  • Cis: Identical groups on the same side
  • Trans: Identical groups on opposite sides
  • E–Z system: Universal system for alkene geometry
  • CIP rules: Priority rules based on atomic number
  • Stereoisomers: Same connectivity, different 3D arrangement
