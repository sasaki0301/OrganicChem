Naming Alkanes

This lesson teaches you how to assign correct IUPAC names to straight-chain and branched alkanes. You will learn how to identify the parent chain, number carbons correctly, name substituents, and assemble the final name.

1. What Are Alkanes?

Alkanes are hydrocarbons containing only single bonds (C–C and C–H). Their names end in -ane.

General formula: CₙH₂ₙ₊₂

Straight-chain alkanes grow by adding CH₂ groups.

Common Parent Chain Names

  • 1 carbon → methane
  • 2 carbons → ethane
  • 3 carbons → propane
  • 4 carbons → butane
  • 5+ carbons → pentane, hexane, heptane, octane...

Activity 1: Quick Check

Name the straight-chain alkane with 7 carbons.

Heptane (7 carbons → hept- + ane).

2. The IUPAC Naming Steps for Branched Alkanes

1. Find the longest continuous carbon chain

This chain becomes the parent chain and determines the base name (meth-, eth-, prop-, pent-, etc.). Always select the chain with the most carbons—even if another one “looks straighter”.

2. Number the chain

Start numbering from the end that gives substituents the lowest possible locants. If a tie occurs, use the next rule (alphabetical priority).

3. Identify substituents

Anything attached to the parent chain is a substituent (methyl, ethyl, propyl, halogens, nitro, etc.). Highlight these groups and note where they are attached.

4. Assign locants and combine

Attach the carbon number to each substituent using a hyphen (e.g., 2-methyl). Separate multiple locants with commas.

5. Add multiplicative prefixes

If substituents repeat, use di-, tri-, tetra- (2,2-dimethyl or 2,3,4-trimethyl). Locants must match the count.

6. Select the suffix

Alkanes always end in -ane. For this lesson, we focus only on simple, saturated hydrocarbons.

7. Combine everything

Final format: Locants + Substituent(s) + Parent + Suffix
E.g., 2,3-dimethylpentane

4. Multiple Substituents

If the same substituent appears more than once, use prefixes:

  • 2 substituents → di-
  • 3 substituents → tri-
  • 4 substituents → tetra-

Correct name: 2,3-dimethylpentane

Activity 4: Identify the Substituents

How many substituents are present in the molecule above?

There are two methyl groups (on carbons 2 and 3).

5. Alphabetical Order

Substituents must be listed in alphabetical order. Prefixes like di-, tri-, sec-, tert- are NOT used for ordering.

Activity 5: Alphabetical Order Check
