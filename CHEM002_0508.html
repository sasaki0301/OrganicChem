Chapter 5 Summary: Alkenes and Alkynes

In this chapter, you explored unsaturated hydrocarbons — compounds containing carbon–carbon double and triple bonds. You learned how structure affects naming, geometry, and isomerism.

1. Alkenes

  • Alkenes contain at least one C=C double bond.
  • General formula: CₙH₂ₙ (acyclic).
  • The double bond restricts rotation, leading to geometric isomerism.
  • Alkenes are named using the suffix -ene.

Checkpoint

Why must the parent chain include the double bond when naming an alkene?

Because the position of the double bond defines the identity of the molecule and determines the suffix (-ene).

2. Cis–Trans and E–Z Isomerism

  • Cis–trans isomerism applies when each carbon of the C=C has one identical substituent.
  • E–Z notation is used when all four substituents are different.
  • Priority is assigned using atomic number (CIP rules).
  • E = opposite sides; Z = same side.

Checkpoint

Why is E–Z notation more general than cis–trans?

Because cis–trans only works when identical groups are present, while E–Z applies to all substituted alkenes.

3. Cycloalkenes

  • Cycloalkenes contain a double bond within a ring.
  • The double bond is assumed to start at carbon 1.
  • Numbering gives substituents the lowest possible locants.
  • Geometric isomerism is common due to ring rigidity.

Checkpoint

Why don’t cycloalkenes freely rotate around the double bond?

Rotation would break the π bond, which requires energy, so the structure is locked.

4. Alkynes

  • Alkynes contain at least one C≡C triple bond.
  • General formula: CₙH₂ₙ₋₂.
  • Triple bonds are linear and highly unsaturated.
  • Named using the suffix -yne.

Checkpoint

Why do alkynes not show cis–trans isomerism?

Triple-bonded carbons are linear, so substituents cannot be arranged on different sides.

5. Mixed Alkene–Alkyne Compounds

  • Contain both a C=C and a C≡C bond.
  • Parent chain must include both multiple bonds.
  • Numbering follows the lowest set of locants rule.
  • If tied, the double bond gets the lower number.
  • Suffix format: -en-yne.

Checkpoint

Which bond gets priority if numbering is ambiguous?

The double bond (ene) receives the lower number.

Key Skills You Should Now Have

  • Identify and classify alkenes, alkynes, and cycloalkenes
  • Correctly number unsaturated carbon chains
  • Apply cis–trans and E–Z rules accurately
  • Name mixed unsaturated hydrocarbons systematically
  • Explain how structure affects isomerism
